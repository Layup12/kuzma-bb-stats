## Описание

REST API сервер для учета статистики

## Установка

```bash
$ yarn add
```

### Для запуска потребуется локально установить postgres

Необходимо создать файл _.[development | production].env_ на базе _.example.env_

## Запуск приложения в dev режиме

```bash

# watch mode
$ yarn start:dev

```
